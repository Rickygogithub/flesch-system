Beautiful Intonation Trainingv1

ピタゴラス音律と純正律に基づいた、開放弦との共鳴を感じながら正確な音程を習得するシステム。調弦・主音ロングトーン・1オクターブ音階・分散和音・チューナー機能を搭載。

調弦

A=442Hzを基準に、完全5度で開放弦を調弦します。
調弦する弦の組み合わせ

A = 442 Hz（基準音・単音）

開放弦の重音

選択した2つの弦を同時に鳴らします。正確な完全5度の響きを確認してください。

周波数の算出（A基準）
A = 442 Hz（基準）
E（ミ） = 442 × (3/2) = 663 Hz
D（レ） = 442 × (2/3) = 294.67 Hz
G（ソ） = 442 × (2/3)² = 196.44 Hz
C（ド） = 442 × (2/3)³ = 130.96 Hz

フラジオにより（差音）を聞き取る練習

弦の真ん中に軽く指を置いて弾くと、基本音ではなく1オクターブ上の音が聞こえます。実音での差音は周波数が低く聞き取りにくいため、フラジオで上げた差音で体感してください。

差音について： 2つの音が正確に揃った時、周波数の差分に対応する「低い音」が聞こえます。これが差音です。
差音についての説明 ▼
差音とは2つの音が正確に揃ったとき、周波数の差分に対応する「低い音」が聞こえる現象です。
各組み合わせの差音周波数：
A/D：442 Hz - 294.67 Hz = 147.33 Hz (D)
D/G：294.67 Hz - 196.44 Hz = 98.23 Hz (G)
G/C：196.44 Hz - 130.96 Hz = 65.48 Hz (C)
実音での差音は周波数が低く聞き取りにくいため、フラジオで1オクターブ上げた差音で体感してください。下の弦の周波数そのものが聞こえます。「この周波数の音が出たら正解」という判定基準になります。
その他の度数での差音周波数：
完全4度：下の弦から完全5度下の音
純正3度：下の弦の1オクターブ下
純正6度：下の弦から完全4度上の音

② 1オクターブ音階（ピタゴラス音律）

主音から1オクターブ、1音4拍で昇降する音階です。
練習のポイント ▼
次の順番で練習すると効果的です。
  1. 音声を聞く（外部の正しい音を、まず受け取る）
  2. 一緒にドレミで歌う（外部の音に合わせて、自分の発声で再現する＝音感の基礎構築）
  3. ツールを止めてドレミで歌う（外部の助けなしで、記憶から再現できるか検証）
  4. ドレミで歌いながら楽器で弾く（発声という認知的基準を保持したまま、楽器という運動出力に接続）
  5. （頭の中でドレミと臨時記号を思い浮かべながら）楽器で弾く（発声という外部化された確認を内面化し、脳内処理だけで運動を制御）
  6. 楽器で音楽的に弾く（技術処理が自動化された状態で、表現に専念）
音声に合わせて弾く場合は音程をピッタリ合わせ、両方の音が溶け合って聞こえなくなるようにしてください。楽器だけで弾く場合は最高音と、1オクターブ下の開放弦の共鳴による振動を最大にし、重音で弾く場合は一つの音に聞こえるようにしてください。
弦（主音）
テンポ 120 BPM

③ 分散和音（純正律）

『ド-ミ♭-ソ（短調）』『ド-ミ-ソ（長調）』（ハ長調の場合）の2つのグループの分散和音の練習。単音モードでは最初と最後に3和音が鳴るので、そのイメージを思い浮かべながら分散和音を弾いてください。
練習のポイント ▼
必要に応じて②のプロセス（の一部）を実践してください。ツールを使わずに弾く際には、最初の音から１オクターブ上の音を弾いている時に開放弦と重音として弾き、１つの音に聞こえるまで音程を合わせてください。
・開放弦を弾くときは、指を押さえないため脳から指への指令が緩みやすくなります。開放弦でも音名を意識して頭の中で鳴らし、回路が細くならないようにしてください。
・5つの三和音（主和音・転回形）は、音程が3度と4度の組み合わせのみで構成されています。音名だけで覚えるより、「どこが3度で、どこが4度か」を意識すると、認知負荷が下がります。
・慣れてきたら、開放弦（C・G・D・A）を主音として、分散和音を続けてビブラートをかけながら弾いてみてください。目的は作曲ではなく、これまで個別に磨いてきた各調性を、実際に移り変わりながら体感することです。曲には転調がつきものなので、この練習で「調から調へ移る感覚」そのものに慣れておくと役立ちます。例えば、G(主音)→D(Gの属調へ)→C(Gの下属調へ、意外な展開に感じるが実は近親調)→G(主調に戻り終止)、という進行。
・長調・短調を固定せず、進行の途中で自由に混ぜてみるのも面白い発見があります（例：C(長)→G(長)→Dm(短)→Am(短)→G(長)→C(長)）。5度進行という骨格を保ったまま、色彩を自由に変える、この「遊び」自体が、実は本質的な練習になります。

構成音の周波数（純正律、ハ長調の例）：
  • C（基本）= 130.96 Hz
  • E（Cの純正3度上、比率5/4）= 130.96 × (5/4) = 163.7 Hz
  • G（Cの完全5度上、比率3/2）= 130.96 × (3/2) = 196.44 Hz
  • C（1オクターブ上）= 130.96 × 2 = 261.92 Hz

※Eの音はCからの純正3度（5/4）で算出しているため、開放E線（Aとの完全5度で調弦）の音程とは異なります。「シントニックコンマ」というズレが生じます。
